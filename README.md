I bridge the gap between **complex hardware systems, artificial intelligence, and scalable full-stack web applications**. From building physical IoT devices to engineering automated SaaS platforms, I love turning ambitious engineering challenges into elegant, real-world solutions.

### 🚀 What I'm Building
- **[Lucente Automation](https://github.com/Bhanu-teja-VCE/lucente-automation)**: An AI-powered B2B lead generation and scraping SaaS platform utilizing intelligent agents.
- **[Focus Lamp](https://github.com/Bhanu-teja-VCE/FocusLamp)**: A custom-built hardware/IoT smart lamp that tracks and boosts productivity using embedded C++ and sensors.
- **[Study Spark 3.0](https://github.com/Bhanu-teja-VCE/the-upgraded-study-spark-3.0)**: An intelligent study assistant leveraging generative AI to supercharge student learning.
- **[Finger Racer](https://github.com/Bhanu-teja-VCE/the-camera-controlled-car)**: A real-time computer vision gesture-controlled racing simulator powered by MediaPipe and p5.js.
